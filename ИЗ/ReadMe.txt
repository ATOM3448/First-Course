Если вы будете опираться на данные файлы при выполнении своих ИЗ, то нужно знать, что нужно исправить некоторые ошибки(мне это делать лень, так ещё и выкладывать).
1. В дракон схеме цикл for оформлен не правильно. В верхней части необходимо объялять и счётчик и условие выхода и увеличение счётчика. Снизу нужно писать "Выход из цикла по переменоой <Ваша переменная счётчика>".
2. В каждой программе, где используется функция rand() необходимо писать следующее: "srand(LC_TIME, "NULL");" для разных результатов рандома при одинаковых параметрах.
3. В каждой программе необходимо писать в конце getch() для того, чтобы exe файл не закрывался сразу после выполнения программы.
